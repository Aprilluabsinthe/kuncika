పారామితులు కుండలీకరణాలు '(' మరియు ')' అక్షరాలలో చేర్చబడ్డాయి. ఐచ్ఛిక పారామితులు మరియు వేరుచేసే కామాలతో చదరపు బ్రాకెట్ '[' మరియు ']' అక్షరాలతో జతచేయబడతాయి (చదరపు బ్రాకెట్లను నమోదు చేయకూడదు).

ఉదాహరణలు చూపించబడ్డాయి bold typewriter font. ప్రతి ఉదాహరణ అది ఏమి చేస్తుందో చూడటానికి దాన్ని వెంటనే అమలు చేయడానికి క్లిక్ చేయవచ్చు.

కాన్వాస్

నేను బటన్ చెల్లింపులో టెక్స్ట్ సహాయం పనిచేస్తుంటుంది.

కుంచిక గ్రాఫిక్స్ గీసిన చోట కాన్వాస్ ఉంటుంది. కుంచిక దాని ప్రయాణ దిశలో సూచించే ఆకుపచ్చ త్రిభుజం.

కమాండ్ ప్రాంతం అంటే ENTER (రిటర్న్) కీని నొక్కడం ద్వారా టెక్స్ట్ ఎంటర్ చేసి అమలు చేయవచ్చు.

The Reset button clears the canvas and moves the turtle to its start position.

Run Demo button runs the function in the Code area named demo().

The Stop button stops animation.

The Save Canvas button takes a snap shot of the canvas and saves in the browser download file with the name TurtleGraphic.png.
